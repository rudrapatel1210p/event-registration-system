🎉 Event Registration System

A modern and interactive Event Registration Web App built using HTML, CSS, and JavaScript. This project allows users to select events, register, and view their registrations in a visually appealing UI.


🎯 Key Features

✨ Elegant UI/UX

Dark modern theme with gradients & glow effects

🧑‍💻 User-Friendly Form

Clean input design with icons
Smart validation

📅 Event Selection System

Interactive event cards
Visual selection highlight

📊 Live Dashboard Stats

Total registrations
Next upcoming event

🚫 Duplicate Protection

Prevents multiple registrations for same event

🏷 Registered Event Pills

Stylish badges for selected events

⚡ Real-Time Updates

No page reload required

🌐 Backend Ready

Easily connect to API (fetch integration)
